# Serie 2

1. Una cola nos permite separar los componentes de una aplicacion y procesar tareas de forma asincrona.

## Ejemplo
Un usuario compra, el sistema guarda el pedido, luego envia un mensaje a cola para que sepa que procesar luego se lee la cola y envia la confirmacion de que el pedido se realizo con exito. 

Esto hace que apesar de tener muchisimos pedidos al mismo tiempo sigue respondiendo rapido mientras que Sin colas el sistema se volveria lento.
